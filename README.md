# SmartFarm Frontend

Selamat datang di repositori frontend SmartFarm! Proyek ini adalah antarmuka pengguna untuk aplikasi SmartFarm, yang dirancang untuk memvisualisasikan data dan interaksi pengguna.

## Deskripsi Proyek

Proyek ini merupakan bagian frontend dari sistem SmartFarm. Ini mencakup halaman web statis yang menampilkan informasi terkait pertanian cerdas. File-file seperti `index.html`, `favicon.png`, `logo.png`, `logo1.png`, `petani-female.jpeg`, dan `petani-male.jpg` menunjukkan bahwa ini adalah proyek berbasis web yang sederhana, kemungkinan besar menggunakan HTML, CSS, dan JavaScript untuk fungsionalitasnya.

![Screenshot Aplikasi](Screenshot1.jpg)

## Cara Menjalankan Proyek

Untuk melihat proyek ini, Anda hanya perlu membuka file `index.html` di browser web pilihan Anda.

1. Navigasi ke direktori proyek.
2. Klik dua kali pada file `index.html` atau buka dengan browser web Anda (misalnya, Chrome, Firefox, Edge).

## Teknologi yang Digunakan

*   **HTML**: Untuk struktur dasar halaman web.
*   **CSS**: Untuk gaya dan tata letak visual.
*   **JavaScript**: (Diasumsikan) Untuk interaktivitas dan fungsionalitas dinamis.
